Gutenberg Blocks to help you with the new WordPress block editor.

== Description ==

DuoGeek Blocks is a collection of page building blocks for the new [Gutenberg](#) block editor. Building pages with the block editor and DuoGeek Blocks gives you more control to quickly create and launch any kind of site you want!

Installing the customizable DuoGeek Block plugin adds a collection of beautiful, site-building blocks to help you customize page layouts, increase engagement, and get results for your business. DuoGeek Blocks provides everything from customizable buttons, to beautifully-designed page sections and full-page layout designs via the Section & Layout block.

Along with the content blocks you'll find in DuoGeek Blocks, we're also publishing helpful articles and tutorials to help you get started with Gutenberg.

### DuoGeek Blocks currently includes the following blocks:

* Advanced Columns Block
* Post Grid Block
* Container Block
* Testimonial Block
* Share Icons Block
* Call-To-Action Block
* Button Block
* Spacer & Divider Block
* Author Profile Block
* Drop Cap Block

### Google AMP Support

The Accelerated Mobile Pages (AMP) project is a publishing format created by Google to enhance site performance for mobile website users. AMP pages are specially designed for Google search users to quickly load website pages without using any extraneous data. DuoGeek Blocks has support for AMP built into each block!

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New.
2. Type "DuoGeek Blocks" into the Search and hit Enter.
3. Locate the DuoGeek Blocks plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will be redirected to the DuoGeek Blocks Getting Started page.

It can also be installed manually.

1. Download the DuoGeek Blocks plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate DuoGeek Blocks in the list and click the *Activate* link.
5. You will be redirected to the DuoGeek Blocks Getting Started page.

== Frequently Asked Questions ==

= Can DuoGeek Blocks be used with any theme? =

Yes, you can use DuoGeek Blocks with any theme.

= Do I need the new block editor to use DuoGeek Blocks? =

Yes, you will need to have WordPress 5.0 or later installed to take advantage of DuoGeek Blocks.
